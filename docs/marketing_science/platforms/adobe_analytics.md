# Key Digital Analytics concepts in Adobe Analytics
- Analytics Report consist of 
	- Dimensions: comprise of values or Items. eg days of the week like SUN, MON etc.
	- Metrics: A metric is the number applied to dimensions. answers like How many/much?
	- Segment: Slice of data that you define to focus on a subset of users. This comprise of Dimensions, Metrics etc.
	- Page Views: Count of the number of times a page has been viewed.
	- Visits: comprise 1 or more page views.
	- Visitors: Count of 1 or more visits tied to a single device.

- Seq2Seq
- a type of neural network that uses deep learning techniques to enable machines to understand and generate natural language.
- consist of an encoder and a decoder, which convert one sequence of data into another.
- This type of network is often used in 
	- machine translation, where the encoder takes the source language and converts it into a vector representation, which the decoder then uses to generate the corresponding text in the target language.
	- develop state-of-the-art machine translation systems, such as [Google Translate](https://www.analyticsvidhya.com/blog/2023/02/model-behind-google-translate-seq2seq-in-machine-learning/) and DeepL.
	- summarization, where the encoder takes a long document and generates a shorter summary. These models have also been used in chatbots and other conversational agents to generate responses to user input.
	- Conversation Systems.
- Seq2Seq models work by first encoding the input sequence into a fixed-length vector representation, which captures the meaning of the sequence. The decoder then uses this vector to generate the output sequence one element at a time, predicting the next element based on the previous one and the context vector.

> A type of machine learning that uses a structure like human brain to breakdown massive datasets.
- Inspired by the human brains neural network.
- Made up of many interconnected nodes or neurons that can learn to perform tasks by processing data and making predictions.
- One of the greatest strength of Artificial Neural Network is they are self tuning, means they are like the musical instrument which are self tuned to get the perfect node. while tuning the gutiar we need to twist the nob but in the neural network tunes their weight automatically.
- ANNs are mostly used for supervised learning.
- layers move from left to right:
	- input --> hidden --> output.
	- if network has lot of hidden layers then it is called deep learning Artificial Neural network.
	- output layer has the probability score.

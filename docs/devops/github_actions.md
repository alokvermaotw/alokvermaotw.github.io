# What is GitHub Actions?
- Automation service for repository related processes.
- Its just like another CI/CD tool like jenkins.

1. Continuous Integration
    - Code Push
    - Unit Test
    - Security Scan
2. Continuous Deployment
    - Build
    - Deploy

## What is Workflow?
- Workflow is a process contains one or more jobs.
- Workflow will be attached to a repository.
- One repository can have multiple workflows.
- Job is just like a task eg. test, build etc

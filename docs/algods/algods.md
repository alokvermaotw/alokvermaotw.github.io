 - Characteristic of Algorithms
	 - Input: 0 or More
	 - Output: At least 1 output
	 - Definition: Should have a clear and concise definition
	 - Finiteness: Duration of the execution should be finite not like sql, web server which runs infinitely.
	 - Effective: not doing anything unnecessary which are not important

- How to write an Algorithm
```c
Algorithm Swap(a, b)
Begin
	temp := a;
	a <- b;
	b = temp;
end
```

- How to analyze an Algorithms
	- Time: how much time it is taking, lower is better and the execution should be faster and efficient
	- Space: 
	- Network Consumption or data Transfer: how much
	- Power consumption
	- CPU Registers consumption

# Introduction to Pipelines and Steps
- ZenML follows a pipeline-based approach to organize machine learning workflow. This method promotes efficiency, repeatability, and collaboration in your projects.
- Think of pipeline like a movie production process. A pipeline is a high level workflow that organizes a series of tasks (steps) required to create a final product. In the context of movie, these steps might include scriptwriting, casting, filming, editing, and distribution.
- Similarly, In ZenML, a Pipeline represents a complete ML workflow. Each step of this pipeline might involve data preparation, feature engineering, model training, model evaluation, and finally, model deployment
- Each step in the pipeline depends on the outputs of the previous ones. For example, you can't start filming without a script, and you can't edit without the filmed footage.

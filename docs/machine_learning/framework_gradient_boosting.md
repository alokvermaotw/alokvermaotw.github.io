

# LightGBM
- **LightGBM** is a gradient-boosting framework based on decision trees to increase the efficiency of the model and reduces memory usage.
- It uses two novel techniques:
	- **Gradient-based One Side Sampling(GOSS)**
		- 
	-  **Exclusive Feature Bundling (EFB)**
- Renowned for its efficiency and effectiveness in handling large datasets and high-dimensional features.
- Ability to incorporate various Regularization techniques that help prevent overfitting and improve model generalization.
- Regularization parameters in LightGBM
	- reg_alpha: L1_Regularization

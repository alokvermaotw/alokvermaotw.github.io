# Regression

## Assumptions of Regression
- _Linear Relationship_: there should have linear relationship between Independent variable and the dependent variable.
- _Independence_: The Residuals are independent. Means there should not have correlation between consecutive residuals in time series data.
- _Homoscedasticity_: The residuals have constant variance at every level of x.
- _Normality of Errors_: The residuals of the model are normally distributed.
